Compress overlay textures in RAM=Comprimir texturas de sobreposição na RAM
Reduces RAM usage to about {{A}}th at the cost of lower quality. Use when loading lots of characters with overlays if you're running out of memory.=Reduz o uso de RAM em cerca de {{A}} ao custo de menor qualidade. Use ao carregar muitos personagens com sobreposições se você estiver ficando sem memória.

Overlay export/open folder=Pasta de exportar/abrir sobreposições

The value needs to be a valid full path to an existing folder. Default folder will be used if the value is invalid. Exported overlays will be saved there, and by default open overlay dialog will show this directory.=O valor precisa ser um caminho completo válido para uma pasta existente. A pasta padrão será usada se o valor for inválido. As sobreposições exportadas serão salvas aqui e, por padrão, a caixa de diálogo Abrir Sobreposição mostrará essa pasta.
