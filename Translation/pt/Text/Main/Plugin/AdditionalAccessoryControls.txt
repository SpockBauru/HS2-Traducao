Slot Overrides - Allows slots on this coordinate to count as other slots for purposes of accessory visibility rules.=Substituições de slot: Permite slots dessa carta contarem como outros slots, para visibilidade de acessórios

All rules count as wildcards, they match any state. Overriding a slot to a different slot means it no longer counts as the original slot. If you want to it to still count as it's original slot, override each of the states of the slot to themselves.=Todas as regras contam como curingas, elas correspondem a qualquer estado. Substituir um slot para um slot diferente significa que ele não conta mais como o slot original. Se você quiser ainda contar como o slot original, substitua cada um dos estados do slot para si.
